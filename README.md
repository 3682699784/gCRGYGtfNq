# Python成都二手房推荐可视化系统

## 核心功能
- 登录注册
- 二手房信息推荐
- 系统简介
- 系统公告
- 二手房信息管理
- 用户管理
- 个人中心
- 系统管理

## 适用场景
- 二手房市场分析
- 房地产从业者信息展示
- 用户查询二手房信息
- 系统维护与管理

## 技术特点
- 使用Django框架实现后端逻辑
- 利用Vue框架构建前端界面
- 数据存储采用MySQL数据库
- 支持远程调试功能
